# Document to Calendar Event Extension
By: Daniel Burnayev

A Chrome Extension pop-up that allows one to submit an image file or screenshot of their tab, and have events determined and added/inserted into one's Google Calendar. This project is still a work in progress and doesn't have a fully released version ready for usage beyond local environments. Go to the local branch to see a currently working and usable version of the extension that can be used locally.

## Required Technologies (when released):
* A Google Account
